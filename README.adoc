= Opinion Account

Opinion Account packages shared account-domain primitives, event payloads, and
integration helpers that power the Opinion platform’s account-facing services.
The repository hosts reusable components so other services can depend on
consistent models, validation, and infrastructure wiring across the account
domain.

== Modules

* `account-common` – foundational value objects, DTOs, codecs, and supporting
  utilities reused by account-centric microservices.

== Requirements

* Java 21 (Temurin 21 or newer JDK).
* Maven Wrapper (`./mvnw`) included with this repository.

== Add as a Dependency

[source,xml]
----
<dependency>
    <groupId>com.inqwise.opinion</groupId>
    <artifactId>opinion-account-common</artifactId>
    <version>${revision}</version>
</dependency>
----

Replace `${revision}` with the released version you wish to consume.

== Development

[subs="+quotes"]
----
# Compile sources #
./mvnw clean compile

# Run the unit test suite #
./mvnw clean test

# Produce source and javadoc jars #
./mvnw clean package source:jar javadoc:jar
----
